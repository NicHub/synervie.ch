1.  Éditer le script JSON-LD ici :
    <https://github.com/NicHub/synervie.ch/blob/main/synervie.jsonld.html>
1.  Vérifier le script JSON-LD sur schema.org<br>
    <https://validator.schema.org/#url=https%3A%2F%2Fnichub.github.io%2Fsynervie.ch%2Fsynervie.jsonld.html>
1.  Vérifier le script JSON-LD sur Google Search<br>
    Aller à <https://search.google.com/test/rich-results><br>
    et coller le lien vers le fichier HTML qui contient le script<br>
    _https://nichub.github.io/synervie.ch/synervie.jsonld.html_
